## :wrench: Installation 

- Clone repo and navigate to project
```
git clone https://github.com/dchima/sparrow.git && cd sparrow
```
- Install latest version of node
```
nvm install
```
- Install dependencies
```
npm i
```
- Run development server
```
npm start
```
